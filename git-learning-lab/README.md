# Git Learning Lab

Este projeto é um laboratório para ajudar um desenvolvedor em formação no aprendizado de Git e GitHub.

## Estrutura do Projeto

- `index.html`: Contém a estrutura da página web.
- `styles/style.css`: Contém os estilos CSS para a apresentação visual da página.
- `README.md`: Documentação do projeto.

## Como Usar

1. Clone o repositório usando o comando:
   ```
   git clone <URL do repositório>
   ```
2. Navegue até o diretório do projeto:
   ```
   cd git-learning-lab
   ```
3. Abra o arquivo `index.html` em um navegador para visualizar a página. Recomendo o uso da extensão Live Server, que te permite visualizar as modificações em tempo real, sem precisar atualizar a página no navegador.
4. Siga as tasks que vou sugerir por meio do Trello ou GitHub Projects - a definir.
5. Divirta-se e bons estudos!
